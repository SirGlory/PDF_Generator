Title: PDF Generator

Description:
This app generates a PDF invoice document from user inputs. 
Accepts customer name, as well as three products and there prices. 
This info, with the total and current day's date are  displayed on a PDF document. 
Names file according to 'date' and 'customer's name'. 
Each run will automatically open the PDF for Windows users.
A second copy, generated and stored on the cloud is created, and its URL outputed

External libraries used: fpdf and filestack. 
Developed in PyCharm with Pyhton.
